This batch file, once run, will create a minecraft server on your desktop, and automatically run it.
Once you run the file wait, for a second Command Line Prompt to appear. This will be your server.
Once it is running you should be able to join the server with the ip of 0, unless you port forward the server.
This was made to quickly and easily set up a Java Minecraft server, without the hassle of messing with different files and
